# Camera Preset Tuning Checklist

이 프로젝트는 이동체 타입별로 `CinematicCameraRig.setPreset(...)`을 호출해 런타임 전환할 수 있습니다.

- `car`: 낮고 약간 뒤에서 따라오는 low chase cam
- `bike`: 조금 더 높고 가까운 추적 시점
- `fallback`: 공통 기본값(안전 프리셋)

## 튜닝 체크리스트

### 1) 저속/중속/고속에서 시야 안정성
- 저속(0~30km/h): 카메라 흔들림이 과하지 않고, 타깃 주변 디테일 확인이 쉬운가?
- 중속(30~90km/h): 오프셋 보간이 부드럽고 튀는 느낌이 없는가?
- 고속(90km/h+): FOV 확장과 카메라 지연이 결합되어도 프레임 중심이 안정적인가?

### 2) 급회전 시 타깃 유지감
- 급회전 순간에도 타깃이 화면 중심 근처에 유지되는가?
- `lookAhead`와 `lookAtOffset` 조합이 시점 붕괴 없이 방향 전환을 예측하는가?
- 과한 spring/damping 세팅으로 턴 후 오버슈팅이 발생하지 않는가?

### 3) 셰이크 과도 여부(멀미)
- 고속 구간에서 `shakeScale`이 과도해 멀미를 유발하지 않는가?
- 저속에서는 셰이크가 거의 느껴지지 않는가?
- 장시간 주행 시 좌우/상하 진동 패턴이 거슬리지 않는가?

### 4) FOV 확장 체감
- 가속 시 개방감이 자연스럽게 증가하는가?
- 감속 시 FOV 복귀가 너무 느리거나 급격하지 않은가?
- 프리셋별(`car`, `bike`) 캐릭터가 시각적으로 구분되는가?
